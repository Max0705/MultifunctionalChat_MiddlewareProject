# MultifunctionalChat_MiddlewareProject
中间件聊天软件。
功能：实时聊天，翻译，图片，传文件，调整字体大小颜色，位置共享。

jms+activemq 实现聊天
api实现翻译、地图

运行界面：
1.一个框：输入用户名和密码（本机IP）
2.界面上显示在线的所有人的和一个群聊
3.点击人或群聊后跳转至此页面，可发消息，可选择接受到的消息的语言，可选择本地文件发送，可位置共享
